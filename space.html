重點規則

  1. 遇到 大括號{} 時，左括號前以及右括號後皆要有一個空格。
  2. 遇到 小括號() 時，左括號前以及右括號後皆要有一個空格。
  3. 遇到 運算子+-*/=.. 時，其前後需加一個空格。
  4. 遇到 分號; 時，其後需加一個空格。
  5. 遇到 逗號, 時，其後需加一個空格。
  6. 遇到 關鍵字if else switch.. 時，其前後需加一個空格。

類別(class)

              class Main extends Site_controller{
              }
            
              class Main extends Site_controller {
              }

函式(function)

              public function index(){
                $this->load_view(null);
              }
            
              public function index () {
                $this->load_view (null);
              }
            
              array_map(function($item){
                return $item;
              },$array);
            
              array_map (function ($item) {
                return $item;
              }, $array);
            
              function func ($a,$b) {
                return $a + $b;
              }

              echo func (1,2);
            
              function func ($a, $b) {
                return $a + $b;
              }

              echo func (1, 2);
